# vue-player
 A simple player base on Vue.js
 一个基于VueJs的简洁音乐播放器


![vue-player](https://github.com/oct16/vue-player/blob/master/vue-player-demo.jpg)

## Install

      拷贝项目
      git clone git@github.com:oct16/vue-player.git
      
      安转依赖
      npm install
      
      开发调试
      npm run dev 
      
      打包部署
      npm run build
